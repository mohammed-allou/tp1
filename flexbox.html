Peut-être! Cela dépend de la façon dont vous envisagez de l’utiliser.
Si vous l’utilisez pour vous référer à GitHub, comme un lien vers GitHub qui dit "Vérifiez-moi sur :octocat:," qui est parfaitement bien.
Toutefois, si vous utilisez l’octocat pour désigner un produit qui n’est pas GitHub, alors ce n’est pas considéré comme une utilisation équitable.
Cela vaut pour toutes les situations d’utilisation, y compris, mais sans s’y limiter, les applications, le matériel d’impression et les sites Web.
